# Checklist do contador responsável

- [ ] UG e competência confirmadas.
- [ ] Fontes e hashes conferidos.
- [ ] Todos os achados possuem evidência.
- [ ] Redação vigente das Macrofunções confirmada.
- [ ] Restrições avaliadas individualmente.
- [ ] Procedimentos de regularização revisados.
- [ ] Nenhum comando transacional foi gerado pelo squad.
- [ ] Evidências pós-regularização anexadas, quando aplicável.
- [ ] Conclusão final registrada pelo contador, fora do squad.
