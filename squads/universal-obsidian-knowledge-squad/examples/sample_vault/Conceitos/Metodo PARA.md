# Metodo PARA

## Definição
PARA é um sistema de organização criado para separar informação por
acionabilidade: Projetos, Áreas, Recursos e Arquivos.

## Aplicação no Obsidian
No Obsidian, cada categoria PARA pode virar uma pasta. Notas conectam-se via
wikilinks, alimentando o grafo do [[Segundo Cerebro]].
