# Dashboard de métricas

- CTR: qualidade de anúncio/chamada.
- CAC: custo de aquisição.
- LTV: valor vitalício.
- ROAS: retorno de mídia.
- Engagement per Post: resposta ao conteúdo.
- NPS: satisfação/indicação.
- Conversion Rate: eficiência do funil.
- AOV: ticket médio.
- CPM: custo de alcance.
- ARPU: receita média por usuário.
