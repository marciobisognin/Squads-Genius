# Source Tree

- `agents/`: papéis especializados do squad.
- `tasks/`: tarefas reutilizáveis.
- `workflows/`: fluxos de execução.
- `scripts/`: CLI e smoke test.
- `templates/`: fichas e planos editáveis.
- `references/`: princípios extraídos do livro-base.
- `examples/`: caso demonstrativo.
- `validation/`: relatórios de validação.
- `.ip/`: autoria e propriedade intelectual.
