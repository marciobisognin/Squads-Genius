# Source tree

- `agents/` — agentes especialistas do Nexum Cognitivo.
- `tasks/` — tarefas operacionais de aprendizagem.
- `workflows/` — fluxo principal YAML.
- `templates/` — modelos de intake, protocolo, métricas e revisão.
- `references/` — análise de fonte e princípios.
- `scripts/` — CLI local de geração/validação.
- `examples/` — exemplo de entrada.
- `generated/` — saídas de demonstração criadas pelo smoke test.
- `validation/` — relatórios de validação.
- `.ip/` — metadados de autoria.
