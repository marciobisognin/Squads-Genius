# AIIA — AI Impact Assessment — SaaS B2B Demonstração

## LLM Support Bot
- Pessoas/grupos afetados: clientes, usuários internos ou candidatos conforme processo.
- Riscos principais: alucinação, vazamento de dados, resposta inadequada.
- Mitigação: revisão humana proporcional, logs, documentação, teste de vieses quando aplicável e canal de contestação.
