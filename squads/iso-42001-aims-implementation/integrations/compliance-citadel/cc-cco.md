# cc-cco — atualização de gestão para AI Governance

O CCO recebe demandas comerciais e contratuais envolvendo IA e delega para `cc-ai-governance-head`, exigindo validação por ética, risco, jurídico e auditoria ISO.

Roteiro: demanda comercial → triagem → AI governance → squad `iso-42001-aims-implementation` → cross-squads LGPD/segurança/jurídico → pacote executivo.
