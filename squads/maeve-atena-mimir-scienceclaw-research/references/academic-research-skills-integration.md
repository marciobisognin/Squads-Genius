# Integração metodológica limpa — Academic Research Skills

## Fonte auditada

- Repositório: https://github.com/Imbad0202/academic-research-skills
- Autor indicado no repositório: Cheng-I Wu
- Licença indicada no repositório: CC BY-NC 4.0
- Escopo: skills para Claude Code voltadas a pesquisa, escrita acadêmica, revisão por pares, auditoria de integridade e orquestração de pipeline.

## Decisão de integração

Este squad permanece **MIT**. Portanto, a integração deste sistema é feita por **adaptação metodológica limpa**, sem copiar textos, prompts, templates, agentes, scripts ou comandos do repositório CC BY-NC 4.0.

O que foi aproveitado em nível conceitual:

1. separar pesquisa, escrita, revisão, revisão de integridade e formatação;
2. exigir checkpoints humanos em decisões substantivas;
3. manter rastreabilidade entre afirmações, evidências e referências;
4. simular revisão por pares com papéis distintos;
5. produzir registro de processo e limitações;
6. bloquear referências inventadas ou não rastreáveis.

O que **não** foi incorporado:

- nenhum arquivo original;
- nenhum prompt literal;
- nenhum script Python/YAML original;
- nenhum comando `/ars-*`;
- nenhum texto de documentação além de identificação, link e licença para atribuição.

## Como citar esta inspiração dentro do squad

Quando necessário, declarar:

> Este workflow foi redesenhado em implementação própria MIT para o ecossistema Maeve/Hermes/OpenSquad, tendo como uma das referências externas de arquitetura o projeto `academic-research-skills` de Cheng-I Wu, licenciado em CC BY-NC 4.0. Não há cópia direta de conteúdo do projeto externo.

## Limite operacional

Se o usuário tiver Claude Code e quiser usar o plugin externo, ele pode fazê-lo separadamente. Este squad, contudo, é autônomo e distribuível sob MIT, pois seus agentes, tarefas, templates e scripts foram reescritos.
