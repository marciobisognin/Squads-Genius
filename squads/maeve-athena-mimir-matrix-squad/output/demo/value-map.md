# Value Map

## Solução
Programa de 4 semanas com microaulas, templates, desafios e apoio por exemplos reais

## Produtos/serviços
- oferta principal;
- materiais de apoio;
- acompanhamento leve.

## Pain relievers
- reduz incerteza
- simplifica aplicação prática
- diminui tempo até o primeiro resultado

## Gain creators
- gera clareza
- aumenta confiança operacional
- cria artefatos reutilizáveis
