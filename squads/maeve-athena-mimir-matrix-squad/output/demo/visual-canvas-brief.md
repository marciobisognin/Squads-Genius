# Visual Canvas Brief

## Layout
Canvas 3 colunas: Cliente → Fit → Valor.

## Cores
- Azul: perfil do cliente.
- Laranja: dores.
- Verde: ganhos.
- Roxo: proposta de valor.
- Amarelo: evidência/teste.

## Componentes
Cards, setas de conexão, matriz de fit e bloco de decisão.
