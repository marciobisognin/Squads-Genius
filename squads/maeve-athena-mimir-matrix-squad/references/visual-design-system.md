# Visual Design System do Squad

## Design Core

- Fundo: dark-neon ou parchment, conforme público.
- Cores funcionais: azul para cliente, verde para ganhos, vermelho/laranja para dores, roxo para valor, amarelo para evidência.
- Tipografia: títulos fortes, corpo legível, labels curtos.
- Componentes: cards, canvas, matriz, timeline, callout, checklist e roteiro de carrossel.

## Materiais visuais gerados

- Canvas visual em blocos.
- Infográfico de fit.
- Carrossel 3:4 para explicação didática.
- Pitch deck curto.
- Prompts para imagens e ícones.

## Regra

O visual precisa melhorar compreensão, hierarquia e tomada de decisão. Não usar decoração sem função.
