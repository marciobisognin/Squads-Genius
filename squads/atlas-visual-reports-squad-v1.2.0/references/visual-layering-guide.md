# Guia de camadas visuais

## Executive Visual Layer
Pouco texto, tese central, KPIs hero, mapa de decisão, riscos/oportunidades e recomendações visuais.

## Analytical Visual Layer
Explica evidências com gráficos, matrizes, fluxos, funis, roadmaps, heatmaps e cards interpretativos.

## Technical Appendix
Preserva metodologia, premissas, tabelas, fontes, dados longos e relatório original em paginação limpa.

## Regra de qualidade
Se o conteúdo principal parece um PDF textual convertido, a etapa falhou. Texto longo deve migrar para o apêndice.
