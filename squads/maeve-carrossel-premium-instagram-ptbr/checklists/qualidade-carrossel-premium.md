# Checklist de Qualidade — Carrossel Premium

- [ ] Briefing interpretado corretamente.
- [ ] Conteúdo em português brasileiro.
- [ ] Narrativa com abertura, desenvolvimento e CTA.
- [ ] Títulos curtos e legíveis em mobile.
- [ ] Tipografia elegante e hierarquia clara.
- [ ] Pelo menos um elemento infográfico por slide.
- [ ] Uso de elementos pixel/ASCII como linguagem visual, sem poluir.
- [ ] Arquivo HTML criado para cada slide.
- [ ] Arquivo PNG criado para cada slide em 1080x1350.
- [ ] `roteiro.md` criado.
- [ ] `legenda.txt` criado.
- [ ] `hashtags.txt` criado.
- [ ] `manifest.json` criado.
- [ ] Todos os materiais na mesma pasta final.
- [ ] Agente de vídeo criou `video/plan.md` e `video/narracao.txt`.
- [ ] Vídeo vertical 1080x1920 criado em `video/final_francisca.mp4`.
- [ ] Voz Francisca usada quando TTS/edge-tts estiver disponível; fallback documentado quando indisponível.
- [ ] Texto do vídeo é grande, premium e visível para Instagram.
