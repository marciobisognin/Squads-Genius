# Task — Construir plano, decisão e context brief

## Processo
1. Gerar decisão: comprar, esperar, praticar, pedir para IA ensinar, pedir para IA executar com contexto ou contratar especialista.
2. Criar plano de prática mínima.
3. Criar context brief para IA cobrir lacunas sem apagar autonomia.
4. Gerar filtro de ROI e compromisso.

## Saída
- Decision Canvas.
- Context Brief para IA.
- Plano básico bem feito.
- Recomendação de investimento.
