# Checklist de Qualidade Artística

- [ ] Pedido do usuário interpretado corretamente
- [ ] Agentes-artistas selecionados com justificativa
- [ ] Composição definida
- [ ] Paleta e luz definidas
- [ ] Técnica pictórica definida
- [ ] Prompt principal detalhado
- [ ] Negative prompt incluído
- [ ] Parâmetros sugeridos
- [ ] Originalidade preservada
- [ ] Selo APROVADO ou REVISAR emitido
