# Mapear todos os elementos

## Objetivo
Listar títulos, textos, objetos, ícones, cores, layout, agrupamentos, metadados e relações.

## Critérios de aceite
- Entrada registrada.
- Saída estruturada em Markdown e JSON quando aplicável.
- Próximo agente recebe contexto suficiente.
- Incertezas marcadas explicitamente.
