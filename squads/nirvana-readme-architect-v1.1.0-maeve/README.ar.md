## نظرة عامة

**Nirvana README Architect** هو فريق من 5 وكلاء متخصصين يعملون في خط أنابيب لتحويل أي قاعدة كود إلى README بمستوى احترافي.

على عكس المولدات البسيطة التي تنتج قوالب عامة، NRA:

- **يحلل** قاعدة الكود الفعلية (المكدس التقني، النصوص، متغيرات البيئة، هيكل المجلدات)
- **يختار** القالب المثالي حسب نوع المشروع (مكتبة، CLI، تطبيق ويب، API، Monorepo، موبايل، فريق)
- **يولّد** المحتوى باستخدام **جميع** ميزات GitHub Flavored Markdown
- **يتحقق** بقائمة مراجعة من 25+ نقطة مع تسجيل تلقائي
- **يصقل** بالشارات، جدول المحتويات، الأقسام القابلة للطي، والتباعد المثالي

> [!TIP]
> الحد الأدنى لدرجة التسليم: **90/100**. يعيد NRA العمل تلقائيًا حتى الوصول لهذا المستوى.

| `nra-content-architect` | **Serif** | Architect | اختيار القوالب وإنشاء المحتوى |
| `nra-quality-validator` | **Lens** | Guardian | التحقق بقائمة مراجعة من 25+ نقطة |

## خط الأنابيب

```
Parse → Scan → Draft → Validate → Polish → Revalidate → Deliver
```

| المرحلة | الوكيل | الوصف |
|:--------|:-------|:------|
| **Parse** | Quill | تحديد المشروع والنوع والنطاق |
| **Scan** | Prism | تحليل عميق لقاعدة الكود |
| **Draft** | Serif | اختيار القالب وإنشاء المحتوى |
| **Validate** | Lens | قائمة مراجعة 25+ نقطة |
| **Polish** | Gloss | جدول المحتويات، الشارات، التباعد |
| **Deliver** | Quill | التسليم مع المقاييس |
