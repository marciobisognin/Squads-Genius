# Tech Stack — dharma-companion

## Runtime
- Synkra AIOS Core >= 2.1.0
- Markdown + YAML (squad definition files)

## Framework
- AIOS Agent System (agents, tasks, workflows)
- Nirvana Squad Creator (meta-tool de geração)

## Domain-Specific
- Tradição Zen-budista Soto (Dogen, Maezumi, linhagem)
- 10 Preceitos Mahayana (moldura ética)
- Prática de zazen (meditação sentada)
- Calendário lunar (ritmo de arrependimento)

## Languages
- Markdown: definições de agentes, tasks, config, READMEs
- YAML: workflows, squad.yaml

## No External Dependencies
- Este squad é puramente baseado em agentes Markdown/YAML
- Não requer pacotes npm, pip ou ferramentas externas
- Opera inteiramente dentro do ecossistema AIOS
