**Goal:** Build a reusable squad for turning recurring problems into context, skills, agents, workflows, and proactive automations.

**Architecture:** Markdown-first squad with agent specs, workflow specs, starter skills, prototype scripts, validation, and packaging.

**Tech Stack:** Markdown, YAML, Python standard library, Hermes skills, cron-ready workflow docs.
