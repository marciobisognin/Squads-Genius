# Context Engineering

## Quando usar
Use para criar arquivos Markdown portáveis de contexto.

## Procedimento
1. Mapear domínios
2. Criar arquivos .md
3. Adicionar exemplos
4. Definir atualização
5. Validar privacidade

## Saída esperada
- Artefato Markdown claro.
- Critérios de qualidade.
- Próximo passo recomendado.
- Indicação de necessidade de aprovação humana quando houver ação externa.
