# NOTICE

**AETHER OS Squad** (`aether-os-squad`) é um squad do repositório Squads-Genius,
construído a partir do PRD "AETHER OS — Sistema Operacional Cognitivo para
Orquestração Autônoma de Squads e Organizações de Agentes" (revisões 1.2 e 1.3),
fornecido por Marcio Bisognin.

## Metodologia

A arquitetura AETHER — Cortex, mentes sintéticas com nomenclatura epistêmica
greco-latina (KRITÉS, BOULÉ, EKLOGÉ, ELENCHUS, TEKMÉRION, THÉMIS, MAIEUTIKÉ,
SÝNTHESIS, MNÉME, AITÍA), motores determinísticos, protocolo SACP, Forja
dinâmica, ciclo de aprendizagem com proveniência, organizações persistentes
(**oikos**, οἶκος) com OIKONÓMOS, perfis de persona (**prósopon**, πρόσωπον)
com o Retratista e a Galeria, contrato de adaptador de host e economia de
tokens neutra — é **metodologia autoral** descrita no PRD do produto. Nenhuma
metodologia de terceiros é copiada ou reembalada. Prósopa derivam apenas de
obra pública e licenciável, com proveniência item a item, rotulagem
obrigatória de conteúdo sintético e bloqueio de personificação.

## Nomenclatura

Os nomes das mentes derivam de raízes gregas clássicas (domínio público) usadas
como **vocabulário temático epistêmico**. "AETHER" referencia o conceito
clássico de éter como camada invisível que conecta tudo. Não há cópia de marca,
arte, texto ou ativo proprietário de terceiros.

## Instrumentos

Instrumentos citados como stack de referência (LangGraph, Pydantic, Langfuse,
DuckDB, Qdrant, FastAPI, pytest etc.) são software de terceiros sob suas
próprias licenças, **consumidos** opcionalmente pelo squad — não redistribuídos.
Os scripts deste squad rodam com **Python 3.11+ stdlib**, sem dependências
externas obrigatórias.

## Fronteira determinística

Modelos de linguagem raciocinam e propõem **somente JSON estruturado**. Todo
cálculo, pontuação, ordenação, classificação de risco, orçamento, quota,
despacho e política de erro é executado por código Python determinístico,
versionado e auditável (`Decimal` para aritmética sensível). Nenhuma decisão
numérica de alto impacto depende da variância de um modelo.
