# Design System — DÉDALO

> A "estética" do DÉDALO é **epistêmica**: clareza, rastreabilidade e separação determinística.
> O artefato visível é o documento (PRD/arquitetura) e o protótipo — não um carrossel.

## Princípios visuais dos entregáveis
| Token | Valor | Uso |
|---|---|---|
| Estrutura | Markdown + tabelas + Mermaid | PRD, arquitetura, fluxo |
| Cor semântica | observado / inferido / hipótese / risco | callouts `> [!NOTE/TIP/IMPORTANT/CAUTION]` |
| Rastreabilidade | matriz fonte→requisito→feature | toda decisão tem origem |
| Determinismo | blocos `python` + `json` | LLM=JSON, Python=cálculo |

## Paleta de codinomes (identidade do squad)
- **Construção/forja:** HÉPHAISTOS (forja), DÉMIOURGÓS (artífice).
- **Discernimento:** HÓROS (fronteira), ELENCHUS (refutação), NÓMOS (lei).
- **Conhecimento:** MNÉMON (memória), LOGISTÉS (cálculo), TÉLOS (finalidade).

## Cor de marca (site)
- Card color sugerido: `#7C3AED` (índigo-violeta — "o fio de Ariadne" do labirinto de Dédalo).

## Tom de voz
- Direto, técnico, sem bajulação. ELENCHUS impõe anti-sycophancy by design.
