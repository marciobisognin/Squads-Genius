# Fontes de incorporação

Este squad combina duas referências técnicas:

1. Atlas Visual Reports Squad v1.2.0 — arquitetura de relatório visual premium, HTML antes do PDF, QA e camadas executiva/analítica/técnica.
2. HyperFrames — composição HTML-native, frames, cenas, atributos de tempo e renderização determinística de vídeo.

A incorporação é arquitetural. Os produtos finais gerados devem ser limpos e não devem expor bastidores de criação.
