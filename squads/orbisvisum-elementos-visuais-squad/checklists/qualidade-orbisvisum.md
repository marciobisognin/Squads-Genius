# Checklist de Qualidade OrbisVisum

- [ ] Todas as imagens foram inventariadas.
- [ ] Todos os textos visíveis foram extraídos ou marcados como ilegíveis.
- [ ] Títulos, categorias, códigos, rodapés, autoria e numeração foram registrados.
- [ ] Cores, layout, tipografia e hierarquia visual foram descritos.
- [ ] A solicitação do usuário foi interpretada antes da solução.
- [ ] Todos os agentes passaram pelo fluxo.
- [ ] O entregável final responde ao problema ou constrói o artefato solicitado.
- [ ] Há rastreabilidade entre imagem, elemento mapeado e decisão final.
- [ ] Resposta final contém licença e autoria.
