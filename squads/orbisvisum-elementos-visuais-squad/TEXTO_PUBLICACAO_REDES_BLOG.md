# Texto de publicação — OrbisVisum

## O que é
O OrbisVisum é um squad premium para transformar imagens complexas em entendimento estruturado e solução prática.

## Para que serve
Serve para analisar prints, infográficos, telas, carrosséis, mapas e documentos visuais. Ele mapeia textos, categorias, cores, layout, hierarquia e relações entre elementos, depois passa por agentes especializados para construir uma resposta ou resolver o problema do usuário.

## O que faz
- Lê a solicitação do usuário.
- Inventaria as imagens.
- Mapeia todos os elementos visuais e textuais.
- Extrai conceitos e organiza uma taxonomia.
- Formula o problema.
- Desenha a solução.
- Constrói o entregável.
- Valida completude e rastreabilidade.

## Exemplo prático
Se o usuário enviar imagens de uma tabela periódica do marketing, o squad identifica páginas, categorias, siglas, conceitos, descrições, estilo visual e transforma tudo em relatório, base de conhecimento, carrossel, sistema de estudos ou blueprint operacional.

## Chamada
Use o OrbisVisum quando uma imagem tiver informação demais para ser apenas “descrita” e precisar virar ação, estrutura ou solução.
