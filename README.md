## Como navegar

- Cada squad possui `squad.yaml` como manifesto principal.
- Os agentes ficam em `agents/`, as tarefas em `tasks/`, os fluxos em `workflows/` e as automações em `scripts/`, quando existirem.
- Cada pasta carregada nesta rodada recebeu um `README.md` próprio com visão geral, uso, estrutura operacional e lista de componentes.
